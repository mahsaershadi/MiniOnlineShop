# MiniOnlineShop

A **mini back-end project** built with **Node.js, Express, and TypeScript**.  
The goal of this project is to practice backend development fundamentals such as building RESTful APIs, organizing project structure, and using TypeScript for type safety.

---

## Features

- Built with **Express.js** and **TypeScript**  
- Simple and modular project structure  
- RESTful API endpoints (CRUD operations)  
- Error handling for invalid requests  
- Easy to extend and maintain  

---

